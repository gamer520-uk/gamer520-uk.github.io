How To Play:
1. Run UWP_Helper.bat, which is located in the game folder. (performed once)
2. Turn on the developer mode (Enable Developer Mode). (we reboot the PC, if necessary) (performed once)
3. Install the game (Install Game). (performed once)
4. Turn off the developer mode (Disable Developer Mode). (performed once)
5. Install the Xbox component from the _CommonRedist folder (otherwise the game will “hang” on the splash screen)
6. Launch the game through the start menu.
